#  Watch Store - Full-Stack E-commerce System

A professional, responsive e-commerce web application for a Watch Store built using **PHP (Native)** and **MySQL**. This project demonstrates a complete order lifecycle, user management, and administrative control.

## 🚀 Key Features
- **User Authentication:** Secure login and registration with session management.
- **Product Management:** Browse products, search by category, and view details.
- **Shopping Cart:** Add, remove, and manage items in the cart with real-time price calculation.
- **Order Lifecycle Tracking:** Visual status updates using icons (Pending ⏳, Paid 💰, Shipped 🚚, Cancelled ❌).
- **Admin Dashboard:** - Manage all customer orders.
  - Advanced search for orders by ID or customer name.
  - Manage system users (View/Delete).
  - Update order status and payment timestamps.

## 🛠️ Technologies Used
- **Backend:** PHP (Native)
- **Database:** MySQL
- **Frontend:** Bootstrap 5, FontAwesome, JavaScript, CSS3
- **Dev Tools:** XAMPP, Git & GitHub
